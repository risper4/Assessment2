# Akan-name-generator
A technical system used to assign Ghananian names to newborns based
 on the user's input onthe date of birth and gender.

 ### Author: Risper Gichia

## Project description
Akan-name-generator is amodern and tech-based system used to calculate a newborn's date of birth and assigning a name based on the day of the week the child is born following the Ghananian culture.The system is meant to assist in easy findings of child's name in a more effecient and effective way.

## Set up instructions
Go to the website page using the url
Download or sign in to the website page
Key in the required date of birth and gender
Acquire the intended name
